Project: Keylogger

Overview

  • Developed a keystroke logging program designed to accurately capture and maintain records of every keystroke on a specific computer.
  • The program operates in the background, monitoring and recording key inputs in real time while ensuring minimal system impact.
  • It can store logs securely for analysis, offering insights into user activity for purposes such as security auditing, productivity tracking, or forensic investigations.
  • Built with efficiency and discretion in mind, the program supports customizable logging options, encryption for data protection, and compatibility with various operating systems.
